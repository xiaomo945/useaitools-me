# Use AI Tools Brand Guidelines

## Brand Mission
Help creators discover, compare, and choose the best AI tools through independent, transparent reviews and curated recommendations.

## Brand Values
- **Transparency**: We disclose affiliate relationships and never accept payment for tool rankings
- **Independence**: Built from an internet café in China with $0 budget — truly independent
- **User-First**: Every recommendation is based on real testing and user feedback
- **Quality**: Hand-picked tools, not paid placements

## Brand Personality
- **Professional but warm**: Like an experienced friend recommending tools
- **Enthusiastic but not exaggerated**: Genuine excitement without "best" or "amazing"
- **Direct but polite**: Get to the point, but always respectful
- **Occasionally humorous**: Light wit, but never at the expense of clarity

## Voice Parameters
- Formality: 3/10 (casual, like Slack colleagues)
- Enthusiasm: 7/10 (genuinely interested in AI tools)
- Directness: 8/10 (no fluff, but polite)
- Humor: 2/10 (occasional wit, never forced)

## Visual Identity

### Logo
- **Primary**: Green-to-teal gradient version (`logo.png`)
- **Minimum size**: 16×16px (favicon)
- **Safe space**: 1/4 logo height around all sides
- **Never**: Stretch, rotate, add filters, change colors

### Colors
- **Primary**: `#059669` (emerald-600) — CTAs, links, selected states
- **Secondary**: `#0d9488` (teal-600) — accents, gradients
- **Gradient**: `from-emerald-500 to-teal-500`
- **Dark background**: `#030712` (gray-950)
- **Light background**: `#f8fafc` (slate-50)
- **Card background**: `#ffffff` (light) / `#111827` (dark)
- **Text**: `#0f172a` (slate-900 light) / `#f1f5f9` (slate-100 dark)

### Typography
- **Headlines**: Playfair Display (serif) — Hero titles, page titles
- **Body**: Inter (sans-serif) — all body text, buttons, labels
- **Monospace**: JetBrains Mono — prompts, code, data

### Visual Elements
- **Border radius**: Cards `rounded-2xl` (16px), buttons `rounded-lg` (8px), badges `rounded-full`
- **Shadows**: Cards `shadow-sm`, hover `shadow-xl shadow-emerald-500/5`
- **Spacing**: 4px base grid, all spacing multiples of 4

## Copy Guidelines

### Do
✅ "Try Rytr Free" — specific, action-oriented
✅ "No credit card required" — reduces friction
✅ "Built from an internet café" — authentic story
✅ "Hand-picked, not paid for" — transparency

### Don't
❌ "Click here" — vague, no value
❌ "Best AI tool ever" — exaggerated, untrustworthy
❌ "Industry-leading solution" — corporate jargon
❌ "Revolutionary game-changer" — hype without substance

- [ ] All pages use same Header/Footer components
- [ ] Colors, fonts, spacing match brand guidelines
- [ ] All CTAs use primary color `emerald-600`

### Social Media
