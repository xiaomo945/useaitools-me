# 🤖 Trae Skills

## Available Skills

### 1. Article Writer (内容生产专家)
- **触发词**: 批量写文章、内容生产、生成文章
- **功能**: 自动化批量生成高质量SEO文章
- **文件**: [article-writer/SKILL.md](article-writer/SKILL.md)

## Usage

When you need to generate content, simply say:
- "批量写10篇文章"
- "内容生产"
- "生成一批AI工具文章"

The skill will automatically:
1. Generate topics based on tools database
2. Create articles with comparison tables
3. Add internal links and CTAs
4. Generate image prompts
5. Embed affiliate tools naturally

## Scripts

- `scripts/batch-content-producer.js` - Main content generation
- `scripts/auto-topic-generator.js` - Topic generation
- `scripts/generate-images.js` - AI image generation
- `scripts/enhance-article-content.js` - Affiliate embedding

## Metrics

- Average: 10 articles in ~5 minutes
- Word count: 1000-1500 per article
- Quality: SEO optimized, comparison tables, internal links
