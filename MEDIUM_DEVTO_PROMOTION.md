- **App Router**: Used the new App Router for better performance
- **Server Components**: Kept things fast by rendering on the server
- **Metadata API**: Great for SEO

- Utility-first approach
- Dark mode support built-in
- Responsive design by default

#### Icons: Lucide React
- Beautiful, consistent icons
- Tree-shakable, keeps bundle small

#### Deployment: GitHub Pages
- Free hosting
- Automatic deployments via GitHub Actions
